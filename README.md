# 🚂 rail-cli-ui

Landing page for [@shan8851/rail-cli](https://www.npmjs.com/package/@shan8851/rail-cli) — UK trains in your terminal.

**🌐 [rail-cli.xyz](https://rail-cli.xyz)** · **📦 [npm](https://www.npmjs.com/package/@shan8851/rail-cli)** · **💻 [CLI source](https://github.com/shan8851/rail-cli)**

## Tech

Vite + React 19 + TypeScript + Tailwind CSS v4. Dark theme, terminal examples, static SPA.

## Local dev

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # output → dist/
```
